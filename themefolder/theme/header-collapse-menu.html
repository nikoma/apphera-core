Collapse Main Menu Hide your main menu in a collapible wrapper.

To add a persistent collapse main menu regardless of the screenwidth simply set/remove the following classes on the following elements:

  • 1 .navbar-toggler : remove any .d-none classes so it shows at all screenwidths.
  • 2 Also ensure .navbar-toggler has the following attributes set: data-toggle="collapse" & data-target=".header-navbar-after" .
  • 3 Within the .header add a wrapper with the "target" class set above and add your menu content within that div: <div class="header-navbar-after bg-primary collapse">Menu content here.....</div> .
